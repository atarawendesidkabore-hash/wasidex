# WAEX Family Characteristics

## Status

- Stage: `proposed_pre_launch_family_layer`
- Purpose: organize country export-index funds inside one regional family

## Core Identity

- Family name: `WAEX Fund Family`
- Full name: `West Africa Export Index Fund Family`
- Family code: `WAEX`
- Base currency: `XOF`
- Region: `West Africa / WAEMU corridor economy`
- Current members: `CIREX`, `BUREX`

## What WAEX Does

- keeps country funds inside one naming system
- keeps upload packages aligned for platforms like `WASI Intelligence`
- allows each country to keep a methodology that fits its real export structure

## Naming System

- `WAEX` = umbrella family
- `CIREX` = Cote d'Ivoire member
- `BUREX` = Burkina Faso member
- Next candidates:
  - `GHAEX`
  - `TOGEX`
  - `BENEX`
  - `MALIEX`
  - `NIGEX`

## Methodology Framework

- Shared core: country raw-material benchmarks measured in `XOF`
- Weighting principle: long-memory physical export-flow weighting
- Reconstitution: `annual`
- Rebalancing: `quarterly`
- Governance: documented index committee and source hierarchy

## Two Operating Models

- Coastal export model:
  - for port-led economies like Cote d'Ivoire
- Landlocked corridor model:
  - for economies like Burkina Faso where export flows depend on border posts, transit routes, and regional corridors
